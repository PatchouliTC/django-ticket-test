## 介绍
使用[loonflow](https://github.com/blackholll/loonflow)作为工作流处理引擎的简易学生请假申请系统

修改自[shutongFlow](https://github.com/youshutong2080/shutongFlow)项目，在该项目上进行的简单二次开发

开发笔记见[开发笔记](/docs/DevelopNote.md)

部署流程见[项目部署](/docs/Run.md)
